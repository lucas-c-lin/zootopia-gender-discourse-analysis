# R Packages Used in Gender Representation Analysis of Zootopia

## Overview
This document lists all R packages used in the quantitative text analysis and sentiment analysis of character discourse in the animated film *Zootopia*, organized by their order of first appearance in the code.

---

## Packages List

### 1. **pacman**
Package management utility that enables convenient loading and installation of multiple packages with a single function call.

### 2. **xml2**
Parses and manipulates XML and HTML documents, enabling extraction of subtitle data from HTML transcripts.

### 3. **rvest**
Web scraping library that extracts structured data from HTML documents using CSS selectors and XPath expressions.

### 4. **dplyr**
Data manipulation framework providing efficient tools for filtering, selecting, mutating, and summarizing data structures.

### 5. **stringr**
String processing library offering consistent functions for detecting, extracting, and replacing text patterns using regular expressions.

### 6. **ggplot2**
Declarative graphics framework for creating publication-quality data visualizations using the grammar of graphics approach.

### 7. **tidytext**
Text mining package that tokenizes documents into word-level data frames and integrates with tidy data analysis workflows.

### 8. **viridis**
Color palette collection providing perceptually uniform and colorblind-friendly color schemes for data visualization.

### 9. **scales**
Utility package for scaling and formatting axis labels, breaks, and color gradients in visualizations and numeric outputs.

### 10. **effsize**
Statistical package calculating effect sizes (Cohen's d, Cliff's delta) to quantify the magnitude of differences between groups.

### 11. **lme4**
Mixed-effects modeling package fitting linear and generalized linear mixed-effects models with random intercepts and slopes.

### 12. **lmerTest**
Extension to lme4 that adds statistical tests (t-tests, F-tests) and p-values for fixed effects in mixed-effects models.

### 13. **dotwhisker**
Visualization package for plotting coefficient estimates and confidence intervals from regression models in a dot-and-whisker format.

### 14. **broom.mixed**
Tidying package that converts mixed-effects model objects into clean data frames for downstream analysis and visualization.

### 15. **textdata**
Utility package providing functions to work with predefined text datasets and lexical resources.

### 16. **sentimentr**
Sentence-level sentiment analysis package that calculates polarity scores while accounting for valence shifters, negations, and intensifiers.

### 17. **syuzhet**
Sentiment analysis package providing access to multiple sentiment dictionaries (including NRC) for emotion classification of text.

### 18. **sjPlot**
Statistical visualization package designed specifically for plotting regression coefficients and mixed-effects model results.

---

## Summary

A total of **18 distinct R packages** were employed in this analysis. These packages collectively enabled: (1) web data extraction and HTML parsing; (2) comprehensive data wrangling and preparation; (3) advanced text tokenization and preprocessing; (4) multifaceted data visualization; (5) statistical hypothesis testing; (6) mixed-effects model specification; (7) word-level and sentence-level sentiment quantification; and (8) emotion classification through multiple lexical resources. The selection reflects a comprehensive computational approach to digital humanities research on gender representation in animated discourse.
